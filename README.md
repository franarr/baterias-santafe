# Baterías Santa Fe — Landing Page

Sitio web oficial de **Baterías Santa Fe**, especialistas en baterías Pioneiro con entrega e instalación a domicilio en Santa Fe y envíos a todo el país.

## 🚀 Stack Tecnológico

- **React 18** + **TypeScript**
- **Vite** (bundler)
- **Tailwind CSS v4** (estilos)
- **Framer Motion** (animaciones)
- **Lucide React** (íconos)

## 📦 Instalación local

```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # build de producción
```

## 📁 Estructura

```
src/components/
├── Layout.tsx        # Header, Footer, MobileNav, WhatsAppFAB
├── Hero.tsx          # Sección principal
├── ValueProps.tsx    # Propuesta de valor
├── Brands.tsx        # Marcas compatibles
├── BatteryFinder.tsx # Buscador de baterías
├── Products.tsx      # Catálogo
├── Services.tsx      # Servicios
├── FAQ.tsx           # Preguntas frecuentes
└── WhatsAppIcon.tsx  # Ícono SVG oficial de WhatsApp
```
